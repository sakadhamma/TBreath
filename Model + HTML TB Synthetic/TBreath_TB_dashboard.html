This dashboard runs on synthetic data, not real patients. The dataset was generated to match the format of a microGC + PID device (retention time, peak area, height, and SNR per VOC). Because synthetic classes are cleanly separable, the model achieves near-perfect cross-validation scores — this demonstrates that the pipeline classifies correctly when given clean signal, but it is not evidence of clinical performance. Validation on real TB patient samples is required before any deployment claim.
